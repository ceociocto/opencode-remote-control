### 为什么 im-hub 更好

| Feature | opencode-remote-control | im-hub |
|---------|------------------------|--------|
| **Supported Tools** | Only OpenCode | OpenCode, Claude Code, and more |
| **Platforms** | Telegram, Feishu, WeChat | **10+ platforms** (Telegram, WhatsApp, Discord, Slack, iMessage, Signal, etc.) |
| **Architecture** | Monolithic | Plugin-based & extensible |
| **Agent Ecosystem** | Limited | Full AI agent support |
| **Active Development** | ❌ Deprecated | ✅ Actively maintained |

### Quick Migration

```bash
# Uninstall old package
npm uninstall -g opencode-remote-control

# Install im-hub
npm install -g im-hub

# Initialize
im-hub init
```
